Programming Lessions:
- Same naming Problematic when mixing xmls from application, utilize prefix
- Process dynamically, never assume completness of Tags (Tree-Level too? use ID-attribute if present)
- case Sensitivity is forced
- entity reference required for chars:
<,>,&,',"
-Comments like this:  <!-- This is a comment -->
- no Carriage return!
- filter for self closing element <element />
- use Xpath to navigate locally in XML (Regex for XML) example /bookstore/book[last()]
- use XLST when displaying XML in Html
- use XQuery when querying XMLs (SQL for XML)
- use Xlink to embed link in elements
- no Errors allowed in XML!!!! Use Validator fopr error resistence.
