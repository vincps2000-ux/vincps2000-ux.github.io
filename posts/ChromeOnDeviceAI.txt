Chrome now exposes built-in AI APIs that run a small foundation model directly on the device. I made an interactive demo with the Summarizer API, which has been available on desktop from Chrome 138.

How to use:
download model if not available using: const availability = await Summarizer.availability();

use create() with a passed options struct, defining parameters of session.

important options:
Typ: Art der zusammenfassung (tldr, teasr, headline, etc..) 
sharedContext : Kontext der abfrage
format: Format of summarized text

after that function summarize can be used for a batched summerisation, and summarizeStreaming for token by token generation.
