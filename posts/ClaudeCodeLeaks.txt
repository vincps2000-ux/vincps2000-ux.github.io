sdk folder:
    contains libraries
    contains CHANGELOG file
    _shim(Intercepts API calls and redirects/changes Parameters. Used to support older apis )

src folder:
    tool Folder:contains tool calls, for various integrations, e.g. Filesystem access via FileReadTool
    command folder: for various actions that change the State of the Application(Login/logout,etc.)
    component folder: for various React component
    constant folder: for Constants, like a claude icon pixelart, or interestingly a black point, that switches to a diffrent char on macos.
    entrypoint: Ways to interact with aplication contains mcp and cli boundarys
    hooks: Code that is executed at certain points in the Application
